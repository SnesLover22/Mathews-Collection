Welcome to the site!

This is a sub site to teach you how to make a bookmarklet

First, copy the code you want (note: No just any JS code can be pasted in, As some I've tried just don't work. All bookmarklets on my page work though.)

Then make a bookmark

Now click out of it

Now click the star again and hit more...

Delete the link in the link area

Now paste in the whole caode (make sure you paste the javascript: part in it)

Now hit save! Note: Some pages, like google product pages (Classroom, gmail, etc.,) don't accept bookmarklets. Google Seach does accept it though.
