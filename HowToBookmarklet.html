Welcome to the site!

This is a "sub site" to teach you how to make a bookmarklet

First, copy the code you want (note: not just any JS code can be pasted in, As some I've tried just don't work. All bookmarklets on my page work though, i've tested. if it doesn't work for you, thats simply a skill issue)

Then make a bookmark

Now click out of it

Now click the star again and hit more...

Delete the link in the link area

Now paste in the whole caode (make sure you paste the javascript: part in it)

Now hit save! Note: Some pages, like Google product pages (Classroom, Gmail, etc.,) don't accept bookmarklets. Google Search pages does accept it though.
